LZW Compression is a text-compression program using the LZW algorithm.

PerceptronNeuralNet is a neural net meant for pattern recognition. We developed it to recognize 7 letters in different fonts. It is coded from scratch in Java. It also contains full JavaDoc documentation which you can open in your browser by opening index.html. You will need to open the folder and rename script.txt to script.js

AnimateQuickSort only executes for one iteration of the quickSort algorithm...not the whole thing

WordCloud is a program which assigns words weight based on how often they are repeated in a sample text.

BlackJack is a JavaFX application, that is pretty self-explanatory.
